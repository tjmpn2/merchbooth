# MerchFlow

Merchandise management system with Square payment integration — inspired by AtVenu.

## Features

- **Dashboard** — Real-time sales metrics, top sellers, transaction history
- **Point of Sale** — Product grid, variant selection, cart management
- **Inventory** — Stock tracking by variant, low-stock alerts, margin calculations
- **Events/Tours** — Multi-venue tour management
- **Settlements** — Artist/venue revenue splits, payout tracking

## Quick Start

```bash
# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Add New Project
3. Import your repo → Deploy
